Important FAQ


What is the Document Object Model(DOM)?
The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, where each HTML element is a node.
When a web page loads, the browser turns the HTML into the DOM tree. JavaScript can then interact with that tree to read, change, add, or remove elements and content
How do you select an element from the DOM?
You can select elements from the DOM using JavaScript DOM methods. Here are the most common ways:
const title = document.getElementById("myTitle"); const items = document.getElementsByClassName("menu-item");
What is event delegation in the context of the DOM, and why is it useful ?
Event delegation is a technique where a single event listener is added to a parent element instead of adding listeners to multiple child elements. The event then bubbles up from the target element to its ancestors, allowing the parent to “catch” the event.

Why is Event Delegation Useful? =Better performance – Fewer event listeners = faster page. = Cleaner code – One listener instead of many. =Dynamic elements supported – Works even for elements added later via JS. = Memory efficient – Uses less memory as fewer listeners are attached.
How do you manipulate an elements attributes and styles using the DOM ?
element.setAttribute("attributeName", "value");
element.getAttribute("attributeName");
element.removeAttribute("attributeName");
img id="myImage" src="old"
let img = document.getElementById("myImage");
// Change the src attribute
img.setAttribute("src", "new");
// Get the current src
console.log(img.getAttribute("src"));
// Remove the src attribute
img.removeAttribute("src");
